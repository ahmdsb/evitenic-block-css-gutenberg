Responsive CSS editor for Gutenberg blocks with custom breakpoints.

== Description ==

This plugin adds a block settings panel in the Gutenberg sidebar. Each block gets a scoped CSS editor with breakpoint presets from the dashboard settings page.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin
3. Go to Settings → Evitenic Block CSS to edit breakpoints

== FAQ ==

= How does selector work? =
Use `selector` in the editor. The plugin replaces it with a unique block wrapper class.

= Can I add more breakpoints? =
Yes. Use the form fields in Settings → Evitenic Block CSS.
